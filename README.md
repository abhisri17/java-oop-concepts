# Java OOP Concepts

A complete Java learning repository covering all four OOP pillars with examples,
interview notes, and a full ATM project.

## Topics Covered

| Folder | Topic | Key Concepts |
|--------|-------|-------------|
| 01-Encapsulation | Encapsulation | Private fields, getters, setters |
| 02-Inheritance | Inheritance | extends, super, method reuse |
| 03-Abstraction | Abstraction | Abstract classes, interfaces |
| 04-Polymorphism | Polymorphism | Overloading, overriding, dynamic dispatch |
| 05-ATM-Project | Full Project | All four OOP concepts combined |

## How to Run

1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/java-oop-concepts.git`
2. Navigate to any folder: `cd 01-Encapsulation`
3. Compile: `javac *.java`
4. Run: `java ClassName`

## Prerequisites

- Java JDK 17+
- VS Code with Java Extension Pack
